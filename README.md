

Introduction:
This project analyzes the Titanic dataset to understand the factors that influenced passenger survival. The goal is to identify patterns and relationships between different variables and survival outcomes.

Dataset Overview:
The dataset contains information about passengers such as age, gender, ticket class, fare, and survival status. It includes both categorical and numerical variables used for analysis.

Methodology:
The data was cleaned by handling missing values and organizing relevant features. Exploratory Data Analysis (EDA) techniques were used to examine patterns and relationships between variables.

Statistical Analysis:
Statistical methods such as mean comparison, survival rate calculation, and correlation analysis were applied. These methods helped identify significant factors affecting survival.

Conclusion:
The analysis shows that factors like gender, passenger class, and age influenced survival chances. The findings provide insights into how different characteristics impacted outcomes during the Titanic disaster.
